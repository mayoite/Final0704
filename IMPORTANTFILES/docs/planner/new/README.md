# New Planner Docs

## Purpose

This folder contains the current planner authority.

If you are deciding what to build, what to trust, or how the planner should evolve, start here.

## Naming

Use product-role names first in this folder.

- `CAD Suite planner app` means `apps/cad-suite`
- `Archived donor planner snapshot` means `07docs/Backupcad`
- `Historical planner docs` means `IMPORTANTFILES/docs/planner/old`
