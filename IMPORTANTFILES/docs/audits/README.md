# Audit Docs

## Purpose

This folder stores audit-style documents: repo audits, review notes, and structured analysis artifacts.

## Files

| File | Role |
|---|---|
| `repo_audit_20_parameters.md` | Broad repo audit across architecture, deployment, state, testing, and platform concerns |

## Use

- Put analysis-heavy documents here.
- Do not mix deployment runbooks or planner design docs into this folder.
