# Diagram Docs

## Purpose

This folder stores standalone diagram artifacts.

These are usually SVGs that explain process, flow, or architecture visually.

## Files

| File | Role |
|---|---|
| `as_is_dfd.svg` | Current-state data flow view |
| `to_be_bpmn.svg` | Target-state BPMN-style view |
| `to_be_bpmn_choices_off.svg` | Target-state BPMN variant |
| `to_be_bpmn_choices_on.svg` | Target-state BPMN variant |

## Use

- Keep visual process artifacts here.
- Planner-specific diagrams that are tightly coupled to planner docs can stay under `planner/`.
